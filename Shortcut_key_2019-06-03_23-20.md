|コマンド|効果|
|:--|:--|
|cmd+p	|ファイルを検索して開く|
|F1	|コマンドパレットを開く|
|cmd + shift + K	|その行を削除|
|alt + ↑(↓)	|その行を上(下)に移動|
|alt + shift + ↑(↓)	|その行を上(下)の行にコピー|
|cmd + ]([)	|インデントの追加(削除)|
|cmd + /	|コメントアウト|
|ctrl + G	|指定行移動|
|ctrl + tab	|編集するファイル(タブ)の移動|
|ctrl+ `	|端末(ターミナル)を起動、コード書く人向け|
|cmd + B	|サイドバーを開く(プロジェクト管理)|
|ctrl + shift + G	|gitのサイドバーを開く|
