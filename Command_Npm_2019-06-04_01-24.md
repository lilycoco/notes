***
### package.json をつくる
	npm init

### サーバーを立てる［エクスプレスモジュールの追加］
	npm instole(i) express

### console.log を起動
	node index.js

### サーバーを立てる
	npm start

### サーバーを止める
	Ctrl+c

### サーバーを立ち続けさせる
	npm i nodemon
	npm run dev

### Swagger起動
	npm run dev:api    

### テストを常に起動させる
	npm run test:watch

### nodemon立ち上げ前に起動させる
	npm run start

### コマンドを追加
	package.json - script に追加する
